# Netflix-tableau-dashboard
## 📊 Objective
To analyze Netflix dataset and identify trends in content distribution, genres, and viewer preferences using Tableau.

## 🛠 Tools Used
- Tableau
- CSV Dataset

## 📁 Project Structure
- workbook/netflix-dashboard.twb
- data/netflix_titles.csv
- images/dashboard.png

## 📈 Key Insights
- Movies account for ~70% of total content on Netflix
- USA contributes the highest number of titles (~40%)
- Most content is rated TV-MA and TV-14
- Content growth peaked after 2015
- Drama and International Movies are top genres
